# BirdHouse Admin Panel


# Project Setup

## -- With Docker Compose
### Build image

```sh
docker-compose build
```

### Start the Docker Container 
```sh
docker-compose up
```

## -- With Yarn

```sh
yarn install
```

### Compile and Hot-Reload for Development

```sh
yarn dev
```

### Type-Check, Compile and Minify for Production

```sh
yarn build
```


### Lint with [ESLint](https://eslint.org/)

```sh
npm run lint
```
